Capabilities

Everything you need to read documents

Built on a real browser engine — no Electron, no heavyweight runtime. Just Go, WebView and native OS APIs.

🎨

GitHub-style Rendering

goldmark with GFM extensions. Tables, task lists, strikethrough, autolinks — all rendered identically to GitHub.

✨

Syntax Highlighting

Code blocks highlighted via chroma with the GitHub Light/Dark palette. Language auto-detection.

∑

Offline KaTeX Math

Full LaTeX math rendering with KaTeX v0.16 — completely embedded, no CDN. Inline $…$, display $$…$$, equation/align environments.

📊

Mermaid Diagrams

Flowcharts, sequence diagrams and more rendered inline. Falls back to code block when offline.

📚

Table of Contents

Auto-generated TOC sidebar (toggle ☰) built from all headings. Shown only when headings are present.

🔍

Full-text Search

Ctrl+F opens an inline search bar. Match highlighting, hit counter and forward/back navigation.

🌙

Three Themes

GitHub Light, GitHub Dark and a retro CRT phosphor-green mode. All preserved across restarts.

📐

Portrait & Landscape

Portrait mode centers content at 750 px (scales with zoom). Landscape uses full window width.

🔎

Zoom (25 %–400 %)

Ctrl+Scroll, Ctrl+/−/0 or toolbar buttons. Zoom level displayed and persisted.

📂

Drag & Drop

Drop any supported file directly onto the window. Native file-picker via Ctrl+O as alternative.

💾

Persistent State

Last opened file, scroll position (per file, FNV-64a hash), zoom, theme and layout — all restored on next launch.

🖥

Native OS Integration

GTK fullscreen (Linux) and WinAPI fullscreen (Windows). Native file dialogs. Correct window icon in taskbar and Alt+Tab.

Installation

Get started in seconds

MD Reader needs no configuration. Download, run, drop a file.

Windows
Linux
Build from Source
  1. Download md-reader-setup.exe from the Releases page.
  2. Run the installer — it creates a Start Menu entry automatically.
  3. Optionally add a desktop shortcut during setup.
  4. Drag any .md, .epub, .fb2, .tex or .txt file onto the window.

Windows 10 or newer recommended (WebView2 is pre-installed). On older systems: install WebView2 Runtime. 

# Install runtime dependencies
sudo apt install libwebkit2gtk-4.1-0 libgtk-3-0

# Download and make executable
wget https://github.com/WaschbaerImHaus/text-reader/releases/latest/download/md-reader
chmod +x md-reader

# Run with a file
./md-reader /path/to/file.md
# Prerequisites
sudo apt install libwebkit2gtk-4.1-dev libgtk-3-dev \
  gcc-mingw-w64-x86-64 g++-mingw-w64-x86-64 nsis

# Clone the repository
git clone https://github.com/WaschbaerImHaus/text-reader.git
cd text-reader

# Build all targets (Linux + Windows + tests)
make all

# Or individual targets
make linux          # Linux x86_64
make windows        # Windows x86_64 (.exe + installer)
make linux-arm64    # Linux ARM64 (requires aarch64-linux-gnu-gcc)
make test           # Run all tests
